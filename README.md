# multi-client-service-marketplace-UCT-
A multi-tenant service marketplace connecting merchants and customers with secure checkout and payments.


# Multi-Client Service Marketplace

## Features
- Merchant onboarding & dashboards
- Customer browsing & checkout
- Secure payment integration
- Reviews & ratings
- Multi-tenant architecture

## Tech Stack
- Node.js + Express
- MySQL
- HTML, CSS, JavaScript
- Stripe Payment Gateway

## System Design
- Multi-tenant architecture
- Role-based authentication
- Secure checkout

## How to Run
1. Clone repo
2. Install dependencies
3. Setup .env
4. Run server
